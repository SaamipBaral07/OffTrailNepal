# OffTrail Nepal

OffTrail Nepal is a tourism platform focused on helping travelers discover trails, guides, and local stays in Nepal.

## Project Structure

- `client/` - React frontend
- `server/` - Node.js + Express backend
- `Increment1/` to `Increment4/` - project increment artifacts and development logs

## Quick Start

### 1) Install dependencies

```bash
cd client && npm install
cd ../server && npm install
```

### 2) Run frontend

```bash
cd client
npm start
```

### 3) Run backend

```bash
cd server
npm run dev
```

## Notes

- Frontend default URL: http://localhost:3000
- Backend runs from `server/src/server.js`
- Configure environment variables for backend before production use
